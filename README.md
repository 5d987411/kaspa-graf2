# Kaspa Graffiti Web UI

A web-based test console for the kaspa-graffiti CLI wallet.

## Features

- **Wallet Management** - Generate, load, and manage multiple wallets
- **Balance Checking** - Query balances from the Kaspa network
- **KAS Transfers** - Send KAS to any address
- **HD Wallets** - Hierarchical deterministic wallet support
- **Network Testing** - Test connection to Kaspa testnet

## Quick Start

```bash
cd src-ui
node server.cjs
# Open http://localhost:8081
```

## Requirements

- Node.js 18+
- kaspa-graffiti-cli binary in the project root
